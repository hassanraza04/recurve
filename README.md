# Recurve

Know your recurring revenue cold.

Recurve connects to a subscription business's Stripe account and turns raw billing
events into the metrics leadership actually runs on — MRR/ARR, the MRR-movement
waterfall, cohort retention, net revenue retention — plus a model that flags the
accounts about to churn. Multi-tenant by design.
