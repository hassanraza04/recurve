# Brand

Recurve's identity is a **precision instrument** — editorial fintech. A financial
terminal with the restraint of a high-end print magazine.

- **Palette:** near-black ink `#0B0B0C`, warm paper `#FAFAF7`, one signal green
  `#16C172` (positive movement + CTAs only), muted amber `#E0A458` (at-risk),
  greys for everything else. (Tokens live in `web/src/app/globals.css`.)
- **Type:** Geist (UI), Geist Mono (every numeral/metric — ledger alignment),
  Fraunces (editorial serif headlines).
- **Logo concept:** a recurve is a bow + a growth curve — a single taut curved
  stroke.

The full identity board (logo system, type specimen, mockups) is generated with
`taste-skill:brandkit` and saved here. **Not yet generated** — that's the first
step of the next work session, before the dashboard UI build (P4) and marketing
site (P6).
